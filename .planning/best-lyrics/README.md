# Best Lyrics — коллекция «лучших стихов»

Растущая коллекция живых результатов пайплайна U→G→C→R, которые признаны
эталонными. Используются как:

1. **Reference baseline** — если новая версия пайплайна делает один из этих
   текстов хуже, что-то сломалось.
2. **A/B корпус для Phase 5** — каждый файл здесь содержит исходный заказ,
   портрет, текст и оценку, чтобы можно было прогнать обновлённую версию
   и сравнить.
3. **Демо для пользователя/инвестора** — конкретные примеры того, что бот
   умеет.

## Формат файла

Каждый файл — `<имя-субъекта>-<версия>.md`, например `zeus-v2.md`.
Внутри: заказ → портрет → доставленный текст → разбор побед и недотягов.
