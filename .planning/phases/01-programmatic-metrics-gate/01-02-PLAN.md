### A. Section parser (Pattern 1, with Pitfall 1 + Pitfall 5 fixes)

- Split lyrics on `\n`, trim each line, skip empty.
- A line matching `/^\[([^\]]+)\]/` opens a new section.
- Normalize section key via case-insensitive prefix match:
  - starts with `припев` → key `'Припев'` (handles `[Припев]`, `[Припев 1]`, `[ПРИПЕВ]`, `[ Припев ]`)
  - starts with `куплет 1` or `куплет1` → `'Куплет 1'`
  - starts with `куплет 2` or `куплет2` → `'Куплет 2'`
  - starts with `бридж` → `'Бридж'`
  - starts with `финал` → `'Финал'`
  - else: trimmed raw value
- For repeated `Припев` blocks (Pitfall 5): keep FIRST occurrence only — repeated chorus blocks are word-for-word identical.
- Returns: `{ 'Куплет 1': [lines], 'Припев': [lines], ... }`.

### B. Syllable counter (Pattern 2)

- Vowel regex MUST be exactly: `/[аеёиоуыэюяАЕЁИОУЫЭЮЯ]/g` (note ё/Ё explicitly listed — Pitfall 4: `[а-я]` does NOT include ё).
- `countSyllables(line)` = `(line.match(VOWEL_REGEX) || []).length`.
- `findChorusSyllableViolations(sections)`:
  - chorusLines = sections['Припев'] || []
  - MAX = 12
  - return lines where count > MAX as `[{line, count, max: 12}, ...]`

- `extractLineFinalWords(lines)`:
  - For each line: trim, split on whitespace, take last token, strip trailing punctuation `[.,!?;:—–\-]`, lowercase.

  - allLines = `Object.values(sections).flat()`.

  - O(n^2) pairwise scan; for `i < j`: if both words have a cluster index AND indices match AND words are NOT identical → record `[w1, w2]`.
  - Deduplicate via key `[w1, w2].sort().join('/')` — return one entry per unique pair.

### D. MATTR-approx (Pattern 4, with Pitfall 3 + Pitfall 4 fixes)

- Tokenize:
  - Lowercase.
  - Strip section tags via `.replace(/\[[^\]]*\]/g, '')`.
  - Split on `/[^а-яёА-ЯЁa-zA-Z0-9]+/` (note ё/Ё explicit — Pitfall 4).
  - Filter empty AND words shorter than 2 chars per `<deep_work_rules>` ("remove words < 2 chars").
- `computeMATTR(tokens, windowSize = 50)`:
  - If `tokens.length < windowSize`: return `unique/total` (full TTR fallback).
  - Else: sliding windows step 1, average `unique/windowSize` over `windowCount = tokens.length - windowSize + 1`.

### E. scoreDraft (Pattern 5, gate function)

```
sections = parseSections(lyrics)
banalePairs = findBanalePairs(sections)
syllableViolations = findChorusSyllableViolations(sections)
tokens = tokenize(lyrics)
diversity = tokens.length > 0 ? computeMATTR(tokens) : 0
skipPipeline = banalePairs.length === 0 && syllableViolations.length === 0 && diversity >= 0.60
return {
  banale_pairs: banalePairs,
  syllable_violations: syllableViolations,
  lexical_diversity: Math.round(diversity * 1000) / 1000,
  skip_pipeline: skipPipeline,
}
```

Constants module-level: `RUSSIAN_VOWELS`, `WORD_BOUNDARY`, `SKIP_PIPELINE_THRESHOLD = 0.60`, `MAX_CHORUS_SYLLABLES = 12`, `MATTR_WINDOW = 50`.

Internal helpers (NOT exported): `parseSections`, `normalizeSectionKey`, `countSyllables`, `findChorusSyllableViolations`, `extractLineFinalWords`, `findBanalePairs`, `tokenize`, `computeMATTR`.
