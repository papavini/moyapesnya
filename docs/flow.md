# Пользовательский флоу (Telegram)

## Основной сценарий

```
/start
  └─► Видео-приветствие + кнопка "Создать свою песню"

Кнопка "Создать" (callback: create)
  └─► Вопрос 1/5: На какое событие?
      [День рождения / Юбилей / Свадьба / Годовщина / ...]
      или "Свой вариант" → ждём текст

  └─► Вопрос 2/5: Жанр
      [Поп / Рок / Рэп / Шансон / Disney / Гитара / Танцевальный]
      или "Свой стиль" → ждём текст

  └─► Вопрос 3/5: Настроение
      [Сентиментальное / Радостное / Романтичное / На наше усмотрение]

  └─► Вопрос 4/5: Голос
      [Мужским / Женским]

  └─► Вопрос 5: Пожелания (текст или голос*)
      Кнопка "Показать пример пожеланий"

  └─► Экран подтверждения: показываем все параметры
      [Создать текст песни] [Внести правки]

  └─► AI генерирует текст (Gemini) — "Сочиняю текст..."

  └─► Показываем текст пользователю
      [Создать песню] [Изменить текст]

      Если "Изменить текст":
        └─► Пользователь пишет правки → AI переделывает → снова показываем

  └─► "Создать песню"
      ├─ PAYWALL_ENABLED=false → сразу генерация
      └─ PAYWALL_ENABLED=true  → экран оплаты Robokassa
                                   [Оплатить N ₽] [Я оплатил] [Назад к тексту]

  └─► Генерация SUNO
      "Проверяю готовность студии…"
      "Отправляю задание в студию…"
      Прогресс-бар: ❤️❤️❤️🖤🖤🖤🖤 42%
      (редактируем одно сообщение, не спамим)

  └─► Готово! Бот отправляет аудиофайл
      "🎵 Название\n\nХотите ещё одну? /start"
```

## Состояния сессии (store.js)

| Состояние | Когда |
|---|---|
| `idle` | Начало / после сброса |
| `awaiting_occasion` | Ждём выбор повода (кнопка) |
| `awaiting_occasion_custom` | Ждём свой вариант повода (текст) |
| `awaiting_genre` | Ждём выбор жанра |
| `awaiting_genre_custom` | Ждём свой стиль (текст) |
| `awaiting_mood` | Ждём выбор настроения |
| `awaiting_voice` | Ждём выбор голоса |
| `awaiting_wishes` | Ждём пожелания (текст/голос) |
| `confirm` | Показан экран подтверждения |
| `review_lyrics` | Пользователь смотрит текст песни |
| `editing_lyrics` | Пользователь вводит правки к тексту |
| `awaiting_payment` | Ждём оплату |
| `generating` | SUNO генерирует трек |

## Команды бота

| Команда | Действие |
|---|---|
| `/start` | Сброс сессии + приветствие с видео |
| `/cancel` | Сброс сессии |
| `/ping` | Проверка доступности студии + кредиты |

## Голосовые сообщения

Пока заглушка — бот просит написать текстом. STT (Speech-to-Text) запланирован.

## VK-бот (упрощённый)

Без AI текстов, без оплаты. Два режима:
- **По описанию** — пользователь описывает трек, SUNO генерирует
- **Свои стихи** — пользователь даёт текст + жанр + название

## Webhook (оплата через Robokassa)

```
pay.vrodnikah.ru/robokassa/result  (Robokassa → сервер)
  └─► verifyResult(MD5 подпись)
  └─► setPaymentStatus('paid')
  └─► runGeneration(lyrics, tags, title)
  └─► bot.api.sendAudio(userId, audioUrl)

pay.vrodnikah.ru/robokassa/success  → redirect → t.me/podaripesniu_bot
pay.vrodnikah.ru/robokassa/fail     → redirect → t.me/podaripesniu_bot
pay.vrodnikah.ru/health             → {"status":"ok"}
```
